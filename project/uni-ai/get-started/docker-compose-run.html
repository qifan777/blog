                  Docker Compose运行

                  获取源码

                  运行DockerCompose

                  cd uni-ai/uni-ai-backend/scrips
docker-compose up -dc

                  导入数据库

                  用可视化工具连接localhost:3308，用户名：root，密码：123456。
